### use tkinter 
```
from tkinter import *
win = Tk()

# Size
win.geometry("1000x500")

# Title
win.title("python-gui-coding")

# Font
win.option_add("*Font", "맑은고딕 25")

# BackGroud Color
win.configure(bg="red")

win.mainloop()
```
